# ЧТО ЭТО ЗА ПРОЕКТ?
{: style="text-align: left" }

**Blitz Classic** - проект, созданный 28 сентярбя 2019 года с одной целью: вернуть «те самые танки» не только в World of Tanks, но и в младшей версии популярной игры.  
Мы крайне достоверно возрождаем старый интерфейс игры, продвигаемся в плане изменения карт под старый лад и эксперементируем с SD-моделями техники.  
Сейчас проект только набирает свои обороты, но вы всегда можете принять участие в разработке, показав свои работы в наших социальных сетях. 
{: style="text-align: left; margin-top: -25px; width: 635px; padding-left: 15px; border-left: 2px solid #f49511;" }

# ИЗ ЧЕГО ОН СОСТОИТ?
{: style="text-align: right" }

Всего есть три типа модификаций. **Интерфейсные**, **графические** и **звуковые**.  
**Интерфейс** улучшает: экран входа в игру, а также интерфейс ангара и боя.  
**Графические** модификации преображают: игровые локации и технику.  
**Звуки** делают ностальгичнее: музыку, технику и окружение.   
{: style="text-align: right;width: 635px;margin: -25px 0 15px auto; padding-right: 15px; border-right: 2px solid #f49511;" }

# КАК И ГДЕ МОЖНО ИГРАТЬ?
{: style="text-align: left" }

Ностальгируйте везде. Проект доступен на Android-устройствах и Windows только для игроков Lesta. Для танкистов из EU-региона возможна игра только на ПК.

## ИНСТРУКЦИИ

### УСТАНОВКА НА WINDOWS
{: style="text-align: left" }

Все очень просто, скачав архивы выше, просто перенесите папку Data из архива в директорию игры с заменой файлов.
{: style="margin-top: -10px; text-align: left" }

### УСТАНОВКА НА ANDROID
{: style="text-align: left" }

На данный момент у нас нет отдельного APK-файла с заранее установленными модификациями, поэтому предлагаем идеальное решение от наших друзей из ForBlitz.  
{: style="margin-top: -10px; text-align: left" }
1. Установите приложение [ForBlitz](https://forblitz.ru/download-application).
2. Перейдите на страницу установки [Tanks Blitz x ForBlitz Edition](https://forblitz.ru/modded-apk) и, исходя из инструкций, загрузите игру.
3. Скачав архив модификаций для Android-устройств, скопируйте все папки внутри.
3. Копируем их и вставляем по пути "**Android/media/ru.forblitz/externalmods**".
4. Заходим в настройки приложения ForBlitz и нажимаем на переключатель «Включить externalmods».
{: style="margin-top: -10px; text-align: left" }

После выполнения этих простых действий вы можете спокойно ностальгировать на улице, в кафе или во время учебы.
